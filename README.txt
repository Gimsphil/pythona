1) 실행
- D:\PythonA\run_pythona.bat

2) 주요 기능
- 파일 메뉴: 파일 불러오기, 저장, 다른 이름으로 저장
- 폴더 불러오기: 폴더 구조(.py 중심) 트리 표시
- 실행(F5): 현재 파일 저장 후 실행
- 중지(Shift+F5): 실행 중 프로세스 종료
- 버그 > 버그내용: 최근 오류(Traceback) 요약 보기
- 도움말 > 작성자 정보: 작성자 연락처 확인

3) 아이콘 설정
- D:\PythonA\app_icon.png 파일을 프로그램 아이콘으로 자동 사용
- 현재 제공하신 아이콘 이미지를 app_icon.png 이름으로 저장해 두면 즉시 반영

4) 더블클릭 연동(.py)
- D:\PythonA\register_py_click.bat 실행
- 이후 .py 파일 더블클릭 시 PythonA Runner가 파일을 자동 로드/실행

5) 연동 해제
- D:\PythonA\unregister_py_click.bat 실행

6) 외부 편집기와 함께 사용
- 코드 수정: VS Code / Cursor / Antigravity
- 테스트 실행: PythonA Runner(F5)
- 오류 확인: PythonA Runner > 버그내용
