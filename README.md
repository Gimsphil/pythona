# pythona
파이썬 실행기 - 파이썬 파일(.py)을 바로 실행해 볼 수 있는 프로그램

## 사용법

### 대화형 모드 (파일 선택)
```bash
python main.py
```
현재 디렉토리에 있는 `.py` 파일 목록을 보여주고, 번호를 입력해 실행할 파일을 선택합니다.

### 파일 직접 실행
```bash
python main.py 파일.py
```
지정한 `.py` 파일을 바로 실행합니다.

### 인자와 함께 실행
```bash
python main.py 파일.py 인자1 인자2
```
`.py` 파일에 추가 인자를 전달해 실행합니다.

### 도움말
```bash
python main.py --help
```

## 예시

```
$ python main.py
실행할 파이썬 파일을 선택하세요:
  1. ./hello.py
  2. ./script.py
  0. 종료

번호 입력: 1
실행 중: python hello.py
----------------------------------------
Hello, World!
----------------------------------------
```
